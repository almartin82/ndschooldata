# Get available years for directory data

Returns the years available in the ND Insights directory data. Currently
2014-2024 (based on verified URL availability).

## Usage

``` r
get_directory_years()
```

## Value

Integer vector of available end years

## Examples

``` r
get_directory_years()
#>  [1] 2014 2015 2016 2017 2018 2019 2020 2021 2022 2023 2024
```
